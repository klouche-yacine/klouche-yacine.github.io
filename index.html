La conception Web est le processus de création de sites Web. Il englobe plusieurs aspects différents, y compris la mise en page Web, la production de contenu et la conception graphique. Bien que les termes conception et développement Web soient souvent utilisés de manière interchangeable, la conception Web est techniquement un sous-ensemble de la catégorie plus large du développement Web. Les sites Web sont créés à l'aide d'un langage de balisage appelé HTML. Les concepteurs Web créent des pages Web à l'aide de balises HTML qui définissent le contenu et les métadonnées de chaque page. La disposition et l'apparence des éléments d'une page Web sont généralement définies à l'aide de CSS ou de feuilles de style en cascade. Par conséquent, la plupart des sites Web incluent une combinaison de code HTML et CSS qui définit la façon dont chaque page apparaîtra dans un navigateur.

  • HTML est un language qui est utilise pour creé le site web.
  • CSS est un autre language qui est uitilie pour le design du site web .
  • Web design encompasses many different skills and disciplines in the production and maintenance of websites
